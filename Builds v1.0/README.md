🎮 操作說明 (Controls)



【基礎移動】



W / A / S / D：移動角色 / 觀察者鏡頭



Space (空白鍵)：跳躍 / (觀察者模式) 上升



Shift (按住)：衝刺 / (觀察者模式) 下降



滑鼠移動：轉動視角



【互動與隊伍】



F 鍵：互動 (打開門、拉開抽屜、按開關)

⦁	準心對準發光的物件後才能互動。



T 鍵：招募/附身

* 對準可控制的角色 (如紙箱) 按 T，即可將其加入隊伍或進行附身。
  (觀察者模式使用左鍵附身)



Q / E 鍵：切換角色

⦁	在隊伍成員之間切換控制權。



Tab 鍵：開啟輪盤選單

* 透過輪盤直接切換到指定角色。



滑鼠左鍵：技能使用



【技能】 不同的物品有不同的技能：



紙箱 📦(Cardboard)：

⦁	短按：吞入前方物品 / 吐出一個物品。

⦁	長按：一口氣吐出所有庫存物品。

⦁	注意：吞太多重物會變慢！



鬧鐘 ⏰(Alarm Clock)：

⦁	短按：啟動 / 關閉。

⦁	響鈴持續吸引NPC注意。

* 注意：及時關閉技能以免被抓住！



手電 🔦(Flashlight)：

⦁	短按：啟動 / 關閉電源。

⦁	長按：使手電向準星方向照明，可閃瞎NPC。

* 注意：NPC在脫離暈眩後會直接進入警戒！



鑰匙 🔑(Key)：

⦁	短按：對準鎖住的門，進行解鎖 (一次性消耗)。



🎯 遊戲目標



⦁	逃脫：尋找關卡中的**撤離點（綠色粒子特效）**，抵達該處即可過關。

⦁	避免被捕：若被 NPC 抓住，遊戲將直接結束並顯示失敗畫面。



🎯 測試重點 (Gameplay Features)

* 請重點測試 \*\*跳躍的手感\*\* 以及 \*\*聲音誘敵的邏輯\*\* 是否正常。



本次測試包含以下核心機制，請試著體驗看看：



高亮互動系統 (Highlight System)

* 滑鼠掃過可互動物件（門、抽屜、道具）時，物件應該會發光或出現輪廓線。
  確認是否能清楚分辨哪些東西可以按。



物理環境 (Physics Interaction)

⦁	抽屜：試著拉開抽屜。

⦁	門：自動感應門。



視線遮擋處理 (Roof System)

* 當你附身物品時，屋頂應該會正常顯示。
  處於觀察者模式 (Spectator) 時，屋頂應該會隱形但保留陰影 (Shadow Only) 以便觀察。



隊伍管理

⦁	試著招募多個角色，並使用 Q/E 順暢切換，檢查攝影機運鏡是否滑順。



視覺化聲音

* 注意觀察地面！當物品從高處掉落或發生撞擊時，會產生 聲音波紋。
  波紋不僅是視覺效果，而是 NPC 實際會聽到的範圍，請利用這一點來聲東擊西。



警戒系統

* 觀察畫面上的 警戒條 (Alert Meter)，它會隨著 NPC 的警戒程度改變顏色（黃 -> 橙 -> 紅）。
  AI 具備 巡邏 (Patrol)、追捕 (Chase)、檢查噪音 (Investigate) 三種狀態，請小心應對 。

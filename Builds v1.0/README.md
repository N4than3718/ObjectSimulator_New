🎮 操作說明 (Controls)
【基礎移動】

W / A / S / D：移動角色 / 觀察者鏡頭

Space (空白鍵)：跳躍 / (觀察者模式) 上升

Shift (按住)：衝刺 / (觀察者模式) 下降

滑鼠移動：轉動視角

【互動與隊伍】

F 鍵：互動 (打開門、拉開抽屜、按開關)。註：準心對準發光的物件後才能互動。

T 鍵：招募 / 附身。對準可控制的角色按 T，即可將其加入隊伍或進行附身（觀察者模式可直接使用左鍵附身）。

Q / E 鍵：快速切換角色控制權。

Tab 鍵：開啟輪盤選單，直接切換到指定角色。

滑鼠左鍵：使用當前物品技能。


🪄 物品技能 (Skills)
不同的物品有不同的專屬技能，請善用它們來破解難關：

📦 紙箱 (Cardboard)：

短按：吞入前方物品 / 吐出一個物品。

長按：一口氣吐出所有庫存物品。（⚠️注意：吞太多重物移動速度會變慢！）


⏰ 鬧鐘 (Alarm Clock)：

短按：啟動 / 關閉響鈴，持續吸引 NPC 注意。（⚠️注意：請及時關閉以免被抓住！）


🔦 手電筒 (Flashlight)：

短按：開關電源。

長按：向準星方向強光照明，可閃瞎 NPC。（⚠️注意：NPC 視力恢復後會直接進入高度警戒！）


🔑 鑰匙 / 門禁卡 (Key)：

短按 / 推進感應區：對準鎖住的門或感應器進行解鎖。


🔨 大槌 (Hammer)：

蓄力擲出：沿指定方向強力擲出，可用於擊暈 NPC。

物理破壞：砸向總電源箱可引發停電。（⚠️注意：需要從高處跳下或具備足夠的物理撞擊力道才能成功砸毀！）


🎯 遊戲目標
完成任務：注意畫面上方的任務指示，達成條件以推進關卡。

逃脫：尋找關卡中的撤離點（綠色粒子特效），抵達該處即可前往下一關。

避免被捕：若被 NPC 抓住，遊戲將直接結束並顯示失敗畫面。


🔬 測試重點 (Gameplay Features)
本次測試包含以下核心機制，強烈建議戴上耳機並體驗看看：

潛行與警戒系統：觀察畫面上的警戒條，它會隨著 NPC 的警戒程度改變顏色。AI 具備巡邏、追捕、聽聲辨位等狀態，請小心應對。

聽覺與視覺回饋：當物品掉落撞擊時，地面會產生「聲音波紋」，這也是 NPC 實際聽到的範圍；滑鼠掃過可互動物件時會有高亮輪廓。

動態音效系統：體驗不同空間的殘響，以及被守衛逼近時的緊張心跳聲！

物理與環境互動：試著推開抽屜、感應自動門，並測試跳躍手感。附身物品時，屋頂會正常顯示；觀察者模式下屋頂會隱形以便總覽。

無縫接軌體驗：過關時你所操控的物品與狀態，將會直接繼承到下一個關卡中！
